 🚀 Business WordPress Theme
https://ismail-business-exclusive.page.gd/
A modern, responsive business website built with custom WordPress development, Bootstrap, and clean UI/UX design.

---

## 📌 Features

* ✅ Custom Header (2 Navbar System)
* ✅ Hero Section with CTA
* ✅ Services (Custom Post Type)
* ✅ Portfolio / Projects Page
* ✅ Testimonials Page
* ✅ Blog System (SEO Ready)
* ✅ Contact Page (Form + Map)
* ✅ FAQ Page (Accordion)
* ✅ Landing Page (Lead Generation)
* ✅ Legal Pages (Privacy, Terms, Cookies)
* ✅ Dynamic Footer (4 Columns + Menu)

---

## 📂 Installation

1. Download or clone the repository
2. Upload theme to `/wp-content/themes/`
3. Activate theme from WordPress dashboard
4. Import demo content (optional)

---

## 🔧 Setup

* Create menus (Primary & Footer)
* Assign pages to menu
* Set homepage from Settings → Reading
* Add services, blog posts, and testimonials
